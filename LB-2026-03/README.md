# LB-2026-03 — How Industrial Systems Communicate: A Field Guide to OT Protocols and Vendors

A Liscere Briefing. This is an **informational field guide**: it maps how the
layers of an industrial system communicate, which protocols dominate each layer,
and how the major controller vendors relate to those protocols. It is built from
published sources, cited in the briefing's References section.

By design it contains **no experimental evidence and no code**: there is no
`evidence/` or `code/` folder. It has **one figure** (`report/figures/fig1.png`,
the communication-layers diagram), authored externally.

## Contents

- `report/LB-2026-03.md` — the briefing source (Markdown with YAML front-matter).
- `report/figures/fig1.png` — Figure 1, the communication-layers diagram.
- `report/LB-2026-03.pdf` — the published PDF, built from the source with the
