# Research

This repository holds Liscere's research material, organised per Technical Report.

Each report has its own top-level folder named `LTR-XXXX-XX`. A folder contains
the source document, the raw evidence, and the source code for the corresponding
experiment. Keeping these together means a report can be read alongside the data
and code it was built from.

## Layout

```
LTR-XXXX-XX/
  README.md      description of the report and its contents
  <source>       the report's source document (Markdown)
  evidence/      raw evidence produced during the work
  <code>         source code for the experiment
```

## Published documents

The published version of each report is a PDF. Those PDFs live at liscere.com.
This repository holds the source material behind them, not the published PDFs.
