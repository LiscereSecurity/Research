# LB-2026-04 — Security by Default, Legacy by Deployment: The Encryption Gap in the Installed S7 Base

A Liscere Briefing. This briefing documents the gap between Siemens S7's
encryption-by-default posture on current hardware and the readable communication
that still dominates the installed base, and what that gap means for a passive
observer. It is built from published sources, cited in the briefing's References
section.

By design it contains **no experimental evidence and no code**: there is no
`evidence/` or `code/` folder, and **no figures**.
