# LB-2026-05 — Passive and Active in Operational Technology: A Field Guide to the Trade-offs

A Liscere Briefing. This briefing separates the two questions the passive/active
debate usually conflates — discovery and inventory on one axis, continuous
monitoring and detection on the other — and weighs the methods on each against
the constraints that make OT unlike IT. It is built from published sources,
cited in the briefing's References section.

By design it contains **no experimental evidence and no code**: there is no
`evidence/` or `code/` folder, and **no figures or tables**.
