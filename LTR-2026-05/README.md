# LTR-2026-05 — The Encryption of Operational Technology: A Protocol Landscape

A Liscere Technical Report. This is a **landscape study**: it is built entirely
from published sources — peer-reviewed measurement studies, standards documents,
and industry publications — all cited in the report's References section.

By design it contains **no experimental evidence and no code**: there is no
`evidence/` or `code/` folder and no figures. Every load-bearing claim in the
report is labelled by the strength of its evidence (measured, reported,
vendor-claimed, or inferred).
